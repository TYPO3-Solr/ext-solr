Routing
=======

This part describes the routing part of EXT:solr.

The configuration of `routing <https://docs.typo3.org/m/typo3/reference-coreapi/main/en-us/ApiOverview/Routing/AdvancedRoutingConfiguration.html>`_ is part of your site configuration.

.. note::
   Currently the configuration over the `site module <https://docs.typo3.org/m/typo3/reference-coreapi/main/en-us/ApiOverview/SiteHandling/Index.html>`_ is not supported.
